Help

How We Work

This issue tracker is designed for two purposes: for customers to raise any issues they encounter whilst using the UnicornAttractor application, and to suggest new features that they think of.

Any bugs will be fixed for free, as we want rid of them as much as our users do!

Any new features suggested will be left to the users to vote for: once a new feature is proving popular with users we will quote how much it will cost to implement. It will then be open to contribute to. Users can contribute as much or as little (well, as little as Â£5!) as they like. Once the target has been reached, our developers will then spend at least 50% of our time creating the new feature.

Why register?

Register in order to raise bugs, suggest new features, vote on existing bug and feature tickets and contribute towards new features!

Raising Bugs

Firstly, what is a bug? A bug is an error or occurrence that doesn't give you the result you expected. If you are experiencing an issue in the application, please log a ticket here and we will deal with it as soon as possible.

Suggesting Features

If you have any ideas about how we can make the UnicornAttractor app an even better experience, please let us know here.

Voting

When voting for bugs or new features, vote positively () for a bug that you are also experiencing, and for a feature that you would like to see implemented. Vote negatively () for a bug that you do not experience when following the same steps, or for a feature that you don't think is needed.

Make your vote count, you only get one!
